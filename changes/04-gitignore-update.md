# Gitignore Update

## Overview
Added `venv2/frontend/build/` to `.gitignore` to prevent committing compiled frontend build artifacts.

## File Modified

### `.gitignore`
Added one line under the "Frontend build artifacts" section:
```
venv2/frontend/build/
```

This prevents the `build/` directory (generated by `npm run build` / `react-scripts build`) from being tracked in git. Build artifacts are large, environment-specific, and should be regenerated during deployment.
